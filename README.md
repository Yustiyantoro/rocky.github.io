# E-Learning Pemrograman Web 1 - Universitas Pamulang

## Cara Praktik Lokal
1. Clone/extract folder ini
2. Buka `index.html` di browser (tidak perlu server)

## Hosting ke GitHub Pages
1. Buat repo GitHub: `e-learning-unpam`
2. Upload semua file
3. Settings → Pages → Branch: main → Save
4. Akses: `https://username.github.io/e-learning-unpam`
